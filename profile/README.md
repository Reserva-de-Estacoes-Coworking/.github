# 🌎 Sobre a DeskGo

A **DeskGo** é uma organização fictícia criada com o objetivo de desenvolver soluções modernas para ambientes de coworking e escritórios inteligentes.

Nosso foco é criar plataformas que tornem a gestão de espaços mais eficiente, prática e acessível, oferecendo uma melhor experiência para gestores e usuários.

---

# 🎯 Nossa Missão

Facilitar a organização de ambientes colaborativos através da tecnologia.

---

# 👁️ Nossa Visão

Ser referência em soluções digitais para gerenciamento de espaços corporativos e coworkings.

# 🛠️ Principal Projeto

## 📌 DeskGo — Sistema de Reserva de Estações

O **DeskGo** permite que usuários reservem mesas em ambientes compartilhados de maneira rápida e intuitiva.

### ✨ Funcionalidades

- Reserva de mesas
- Calendário de disponibilidade
- Cancelamento de reservas
- Gestão de estações
- Controle de disponibilidade
- Validação inteligente de reservas
